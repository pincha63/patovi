*** Clone new repo to github
Given the local repository - - after a relevant commit:
Login to Github
At the top right of the page, click the '+' icon and then select 'New Repository'
Give your repository a name--ideally the same name as your local project
Click 'Create Repository'
The 'Quick setup (etc.)' screen is shown
Copy the link in the input right beneath the title - - should be something like https://github.com/pincha63/reponame.git 
In the local command line, type: git remote add origin https://github.com/pincha63/reponame.git
Then type: git push origin master
In Github, the repository should now include all the relevant files
Add readme.md and whatever else is needed

*** Deploy Node app to NFS
For deployment of node app to NFS see:
http://web.archive.org/web/20161108020800/http://www.mopsled.com/2015/run-nodejs-on-nearlyfreespeechnet/
but the basics are:
 1. get the repository done
 2. get the site going in NFS
		Note: my_site.nfshost.com gets https but own domains don't
		For that, I'd need a key and TLS etc. (homework)
 3. Go to the Config Information section and on click the Edit button on Server Type
 4. Select the Custom domain type
 5. ssh to ssh.phx.nearlyfreespeech.net using elpincha_sitename and NFS password
 6. cd /home/protected
 7. git clone https://github.com/heroku/my_repo.git
 8. cd my_repo
 9. npm install
10. have a text file in my_repo named run.sh which has the command: npm run start
		Note: more configs can be applied here (homework)
11. From the my_repo directory run: chmod +x run.sh
12. ./run.sh
13. If all is well by now, the app will run at the specified port (e.g. 3200)
14. Press CTRL+X to kill your service on SSH.
15. Note: NFS will use Proxies to give your server access to low level ports.
16. Go to the site screen in NFS, and in the Daemons section, click Add a Daemon
17. Config the daemon:
		Tag: patovi_node (<-- unique per-site tag to ID the daemon)
		Command Line: /home/protected/<NODE_FOLDER>/run.sh
		Working Directory: /home/protected/<NODE_FOLDER>
		Run Daemon As: web
18. Click Add Daemon
19. Go to the site screen in NFS, and in the Proxies section, click Add a Proxy
20. Configure proxy:
		Protocol: http
		Base URI: /
		Document Root: /
		Target Port: 3200 (or whatever I chose)
		Direct: Bypass Apache Entirely (<- must be Checked)
21. Click Add Proxy
22. Check site is live. If not so, check logs under /home/logs/something.log
